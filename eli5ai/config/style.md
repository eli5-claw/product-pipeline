## Formatting Rules

1. **Symbols:**
   - `‚ñ∫` for section headers
   - `‚ñ∏` for bullet points
   - `‚Äî` for dividers (sparingly)
   - `üßµ` in hook only

2. **Capitalization:**
   - Title case for section headers
   - Sentence case for body text

3. **Numbers:**
   - Use K/M/B for large numbers (43K+, $50K)
- Round to 1-2 significant figures

4. **Mentions:**
   - Always @mention relevant projects
   - Include 3-5 credits at end

5. **Tone:**
   - Confident but not hype-y
   - Educational, not promotional
   - Assume reader is smart but new to topic

## Infographic Templates

### Template 1: Definition Card
- Large: "What is [X]?"
- Subtitle: One-line definition
- 3 key features as icons + text

### Template 2: Step Flow
- Horizontal or vertical flow
- 3-5 numbered steps
- Arrow connectors
- Minimal text per step

### Template 3: Ecosystem Map
- Center: Main topic
- Surrounding: Categories with logos/names
- Clean, spaced layout

## Visual Style

- **Background:** #0a0a0a (near black)
- **Text:** #ffffff (white)
- **Accent:** #3b82f6 (blue) or project brand color
- **Font:** Inter or system sans-serif
- **Border radius:** 12px for cards
