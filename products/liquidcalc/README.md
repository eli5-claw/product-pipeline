# LiquidCalc

**Leverage/Liquidation Calculator for Crypto Futures Traders**

Price: $2.99 one-time purchase

## Features

- **Liquidation Calculator**: Calculate exact liquidation prices for isolated and cross margin positions
- **Position Sizing**: Risk-based position sizing with stop loss integration
- **P&L Calculator**: Calculate profits, losses, ROI, and ROE with fee consideration
- **Clean UI**: Trader-focused design with dark mode support

## Project Structure

LiquidCalc/

## Build Instructions

1. Open in Xcode 14+
2. Set your Team ID in Info.json
3. Build for iOS 15.0+
4. Archive and submit to App Store
