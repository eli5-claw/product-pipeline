# Design Movements

See designers.md for specific designer profiles.

## Historical Context

### Modernism (1920s-1960s)
Form follows function. Rejection of ornament. Grid systems.
Key figures: Mies van der Rohe, Le Corbusier, Dieter Rams

### Postmodernism (1970s-1990s)
Playful, referential, breaking rules.
Key figures: Ettore Sottsass, Memphis Group

### Digital Native (2000s-present)
Screen-first, interactive, iterative.
Key figures: Jonathan Ive (early), Stripe design team, Linear team

## Current Trends

### 2024-2025
- Bento box layouts
- Heavy serif display type
- 3D elements with soft lighting
- Micro-interactions
- AI-generated imagery integration

### Anti-Trends (Avoid)
- Generic gradient backgrounds
- Overly rounded corners on everything
- Stock illustrations
- "Not X but Y" formulaic copy
