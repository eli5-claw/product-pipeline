## Design Movements

### Brutalism
Raw, unpolished, anti-aesthetic. Exposed structure, system fonts, high contrast.
Reference: Bloomberg's 2016 web redesign, brutalistwebsites.com

### Neumorphism
Soft UI, extruded plastic. Subtle shadows, monochromatic.
Reference: Apple iOS 13-16, various Dribbble trends

### Glassmorphism
Translucency, blur effects, floating layers.
Reference: Apple macOS Big Sur, Windows 11

### Dark Mode Mastery
Not just inverted colors. Reduced brightness, desaturated accents, depth through elevation.
Reference: Linear, Discord, VS Code
