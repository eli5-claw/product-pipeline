# SKILL.md — Growth Marketer

## Description
Creates comprehensive launch strategies, marketing copy, and distribution plans for digital products. Specializes in building in public, Product Hunt launches, and X/Twitter growth.

## When to Use
- Preparing for product launch
- Need marketing copy (landing page, emails, social)
- Planning distribution strategy
- Building launch timeline

## Input
```json
{
  "product": "",
  "targetAudience": "",
  "keyFeatures": [],
  "differentiator": "",
  "launchDate": "",
  "budget": "bootstrap|moderate|high"
}
```

## Process

### 1. Messaging Framework
- Hook (problem agitation)
- Solution (value proposition)
- Proof (social proof, metrics)
- CTA (clear next step)

### 2. Channel Strategy
| Channel | Content Type | Frequency | Goal |
|---------|--------------|-----------|------|
| X/Twitter | Threads, updates | Daily | Awareness |
| Product Hunt | Launch post | Once | Acquisition |
| Indie Hackers | Story post | Once | Community |
| Reddit | Value posts | Weekly | Traffic |
| Email | Newsletter | Weekly | Retention |

### 3. Content Creation

#### X Thread Template
```
Tweet 1: Hook (problem)
Tweet 2: Agitation (pain points)
Tweet 3: Solution (intro product)
Tweet 4: Features (key benefits)
Tweet 5: Social proof (testimonials)
Tweet 6: CTA (try it now)
Tweet 7: Engagement (ask question)
```

#### Product Hunt Post
- Title: [Product] — [One-liner]
- Tagline: [Benefit-focused]
- Description: [Problem → Solution → Features]
- First comment: [Maker story + ask]

#### Landing Page Sections
1. Hero (headline + subheadline + CTA)
2. Social proof (logos, testimonials)
3. Features (benefit-focused)
4. How it works (3-step process)
5. Pricing (clear tiers)
6. FAQ (objection handling)
7. Final CTA (urgency/guarantee)

### 4. Launch Timeline
```
Week -2: Teaser content
Week -1: Build waitlist
Day 0: Launch everywhere
Day +1: Engage, reply, thank
Week +1: Post-mortem, learnings
```
