## SurrealDB
• **Summary:** Multi-model database unifying document, graph, relational, time-series, and key-value data in a single Rust engine.
• **Link:** https://github.com/surrealdb/surrealdb
• **Category:** Developer Tools / Database
• **Why it matters:** SurrealDB eliminates the need for multiple database systems by combining everything into one engine. It runs embedded (in-app), in the browser via WebAssembly, at the edge, or as a distributed cluster. With built-in real-time APIs, row-level permissions, and support for SQL + GraphQL querying, it dramatically simplifies backend architecture. For AI applications, it serves as an ideal knowledge graph and memory layer — a critical infrastructure piece as agents need persistent, queryable memory systems.
