# Obsidian Vault for Cron Jobs

This vault stores all cron job outputs for easy reference and search.

## Folder Structure

- **01 - Daily Briefing** - Morning & Afternoon Briefing (20+ stories)
- **02 - AI News** - Daily AI News
- **03 - Tech News** - Daily Tech News  
- **04 - On-Chain Analysis** - Daily On-Chain Analysis
- **05 - Crypto Research** - Daily Crypto Research
- **06 - GitHub Explorer** - GitHub Explorer (5 innovative repos, 3x daily)
- **07 - Market Pulse** - Midday & Evening Market Pulse
- **08 - Breaking News** - Overnight & Early Morning Wrap

## Naming Convention

Files: `YYYY-MM-DD HH-MM.md`

Example: `2025-05-22 09-00.md`
